# certificate-checker
Validation SSL certificates from a URL or a local certificate (.crt) file

## Usage
```
certificate-checker <command>

Commands:
  certificate-checker check <locations...>  Check the certificates at the given
                                            location(s).

Options:
      --exit-code  Exit with a non-zero code when certificate validation fails
                                                       [boolean] [default: true]
  -j, --json       Output results in JSON format                       [boolean]
  -v, --verbose    Verbose output                                      [boolean]
      --help       Show help                                           [boolean]
      --version    Show version number                                 [boolean]
```
